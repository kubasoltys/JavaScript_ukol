1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je programovací jazyk, který se často používá na webových stránkách.
  • Je interpretovaný a běží v prohlížeči a umožňuje interakci s webovými stránkami.
  • Používá se k vytváření interaktivních prvků na web. stránkách a k validaci formulářů
  • Dále umožňuje provádět dynamické změny dat na web. stránkách aniž by se načítala celá stránka znovu.
  • Dále se používá k vývoji jednoduchých a také složitých webových her.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Indent code to make it readable. Is recommended to use 2 or 4 spaces for indentation.
  2. Keep code as simple as possible.
  3. Use descriptive variables, function and class names.
  4. Comment the code.
  5. Is necessary to use semicolons at the end of each line.

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>
// 2. Inline JavaScript
<button onclick="alert('Ahoj')">Tlačítko</button>
// 3. Externí JavaScript
<script src="skript.js"></script>
// 4. Externí internetové knihovny a frameworky
<script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>
// 5. Moduly
<script type="module" src="mujModul.js"></script>
// 6. Dynamické načtení skriptů
let skript = document.createElement('script'); skript.src = 'skript.js'; document.body.appendChild(skript);

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %= (Přiřaď a modulo), **= (Přiřaď a umocni), <<= (Přiřaď a bitový posun doleva), >>= (Přiřaď a bitový posun doprava)

Logické

&&, ||, ! (Not - pokud je výraz true, změní ho na false), ? (Ternární operátor)

Porovnávací

== (Porovnává rovnosti), === (Porovnává rovnosti ale i datové typy), != (Porovnává nerovnosti), !== (Porovnává nerovnosti ale i datové typy), <, >, <=, >=

Bitové

& (Pro každý bit v obou číslech vrátí 1, pokud jsou oba bity 1, jinak vrátí 0), | (Pro každý bit v obou číslech vrátí 1, pokud alespoň jeden z bitů je 1. Jinak vrátí 0), ^ (Vrací 1 pro každý bit, kde jsou bity v číslech různé. Jinak vrátí 0), ~ (Invertuje bity v čísle. Každý 0 se stane 1 a naopak), << (Posune bity v čísle doleva o určitý počet pozic. Prázdné pozice na pravé straně jsou zaplněny nulami), >> (Posune bity v čísle doprava o určitý počet pozic. Při posunu doprava jsou prázdné pozice na levé straně zaplněny kopii nejvýznamnějšího bitu)

Typové

typeof (typ proměnné), instanceof (Umožňuje zjistit, zda objekt je instancí určitého datového typu)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Nahoře uprostřed vyskočí popup okno ihned jakmile otevřeme stránku, ve kterém bude napsané 'Nezadal jsi správné heslo!'*/
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt("Zadejte heslo:");

if (password !== null) {
console.log("Zadali jste heslo: " + password);
} else {
console.log("Heslo nebylo zadané.");
}

confirm()

/* vyskočí popup okno, které se vás ptá, zda chtece přejít na stránku seznam.cz, jakmile kliknete na OK tak vás to zde přesměruje. */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Mouhou být obtěžující a rušit uživatele.
Mnoho moderních prohlížečů blokuje popup okna.
Mohou být potencionálním bezpečnostním rizikem. Malware a phishing
Na mobilních zařízeních mohou být nepříjemná.
